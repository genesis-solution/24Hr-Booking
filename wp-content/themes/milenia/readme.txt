== Description ==
Milenia WordPress Theme is a responsive WordPress theme created especially for hotels, resorts, hostels, boutique and apartments a complete hotel and vacation rental booking system within theme package. Developed for the latest WordPress version. Milenia supports all responsive layouts and looks great on all devices. == Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on the "Upload Theme" button.
3. Choose milenia.zip archive form your package and click on the "Install Now" button.
4. Click the "Activate" link after the installation.
5. Read the documentation in your package to get more information how to customize the theme.
