== Description ==

Edit existing or add new checkout fields in Hotel Booking to collect more information about your guests.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
