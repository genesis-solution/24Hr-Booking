== Description ==

Automate common notifications by sending check-in and check-out date-driven emails, such as key pick-up instructions, feedback request, etc.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
